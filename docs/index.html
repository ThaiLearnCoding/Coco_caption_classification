Dataset snapshot and pipeline

We track 10 curated categories and enforce strict k-shot splits for probing.

  • Input: 1 image + 12 candidate captions (1 correct, 11 distractors).
  • Splits: k-shot removal for k={0, 8} preserves test integrity.
  • Normalization: CLIP-specific mean and std for each backbone.
  • Adapters: Residual image and text adapters with L2 normalization and cosine logits.

Built with PyTorch, CLIP, Streamlit, and WandB.
